# My Cheat Sheet for NeoVim

## Customized mappings

### In normal mode

* `<Esc><Esc>`: nohlserach
* `<leader>{y, Y}`: copy to clickboard
* `<leader>{p, P}`: paste from clickboard

### In insert mode

* `jj`: Exit from insert mode
* `j<Space>`: insert "j"

## Mappings for plugins

### hop.nvim

Neovim motions on speed.

* `<leader>w`: Word mode
* `<leader>f`: 1-char mode
* `<leader>s`: 2-char mode
* `<leader>/`: Pattern mode
